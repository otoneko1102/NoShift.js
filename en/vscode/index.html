NoShift.js

VS Code Extension

The official VS Code extension provides first-class editor support for NoShift.js.

Installation

Search for NoShift.js in the VS Code Extensions marketplace, or install directly:

ext install otoneko1102.noshift-vscode

View on Marketplace →

Features

  • Syntax highlighting for .nsjs files
  • Auto-complete for NoShift.js ^ sequences
  • File icon for .nsjs files
  • Code snippets for common patterns
